RestrictedCapacityCollectionTOverwrite メソッド

指定したインデックスを起点として、要素の上書き/追加を行う。

名前空間:  WodiLib.Sys
アセンブリ:  WodiLib (in WodiLib.dll) バージョン: 0.5.3
構文
C#
コピー
public void Overwrite(
	int index,
	IReadOnlyList<T> list
)

パラメーター

index
型: SystemInt32
[Range(0, Count)] インデックス
list
型: System.Collections.GenericIReadOnlyListT
[NotNull] 上書き/追加リスト
例外
例外条件
ArgumentOutOfRangeExceptionindexが指定範囲外の場合
ArgumentNullExceptionlistがnullの場合
ArgumentExceptionlist中にnull要素が含まれる場合
InvalidOperationException追加操作によって要素数がMaxCapacityを超える場合
例
コピー
var target = new List<int> { 0, 1, 2, 3 };
var dst = new List<int> { 10, 11, 12 };
target.Overwrite(2, dst);
// target is { 0, 1, 10, 11, 12 }
コピー
var target = new List<int> { 0, 1, 2, 3 };
var dst = new List<int> { 10 };
target.Overwrite(2, dst);
// target is { 0, 1, 10, 3 }
